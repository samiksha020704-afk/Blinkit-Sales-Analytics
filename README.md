# 🛒 Blinkit Sales Analytics

## 📌 Project Overview

This project analyzes Blinkit's sales data using **SQL** and **Python** to uncover sales trends, evaluate outlet performance, and generate actionable business insights.

---

## 🛠️ Tech Stack

- SQL (MySQL)
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

- Retail sales dataset containing **8,523 records**
- Includes product information, outlet details, sales, ratings, and item visibility.

---

## 📊 Project Workflow

- Data Cleaning
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Business Insights
- Data Visualization

---

## 📈 Key Business Insights

- Fruits and Vegetables generated the highest sales (₹178,124.08).
- Supermarket Type1 contributed the highest total sales (₹787,549.89).
- Medium-sized outlets generated the highest revenue.
- Tier 3 locations achieved the highest sales.
- Meat category received the highest average customer rating.
- Outlets established in 2018 recorded the highest total sales.
